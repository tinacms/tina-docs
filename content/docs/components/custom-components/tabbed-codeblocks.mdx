Tabbed code blocks let you present multiple language variants or configurations side-by-sideâ€”ideal for showing equivalent code in JavaScript, TypeScript, Bash, or even specific queries and responses. This improves readability and UX for polyglot dev environments.

Such like:

<queryResponseTabs
  query="// JavaScript
  fetch('/api/data')
    .then(res => res.json())
    .then(data => console.log(data))
  "
  response="// TypeScript
  async function getData(): Promise<void> {
    const res = await fetch('/api/data')
    const data = await res.json()
    console.log(data)
  }
  "
  customQueryName="JavaScript"
  customResponseName="TypeScript"
/>

or

<queryResponseTabs query="const CONTENT_MANAGEMENT = 'Optimized';" response="const LLAMAS = '100';" />
