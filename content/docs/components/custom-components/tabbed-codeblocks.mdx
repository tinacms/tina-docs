---
title: Tabbed Code Blocks
last_edited: '2025-07-10T01:33:46.150Z'
---

Tabbed code blocks let you present multiple language variants or configurations side-by-side—ideal for showing equivalent code in JavaScript, TypeScript, Bash, or even specific queries and responses. This improves readability and UX for polyglot dev environments.

Such like:

<codeTabs
  tabs={[
    { name: "Test", content: "ome�other�tab" },
    {
      name: "Query",
      content: "const CONTENT_MANAGEMENT = 'Optimized';",
      language: "python"
    },
    { name: "Response", content: "const LLAMAS = '100';", language: "tsx" }
  ]}
  initialSelectedIndex={1}
/>

<codeTabs
  tabs={[
    { name: "Tab", content: "const CONTENT_MANAGEMENT = 'Optimized';" },
    { name: "Query", content: "const CONTENT_MANAGEMENT = 'Optimized';" },
    { name: "Response", content: "const LLAMAS = '100';" }
  ]}
  initialSelectedIndex={2}
/>

<codeTabs
  tabs={[
    {
      name: "Tab",
      content: "const CONTENT_MANAGEMENT = 'Optimized';",
      language: "text"
    },
    {
      name: "Query",
      content: "const CONTENT_MANAGEMENT = 'Optimized';",
      language: "text"
    },
    { name: "Response", content: "const LLAMAS = '100';", language: "text" }
  ]}
  initialSelectedIndex={0}
/>
