This endpoint allows you to upload an image file for a specific pet in your store. Images will be associated with the pet identified by the `petId` parameter.

## Endpoint Details

**Method:** `POST`

**Path:** `/pet/{petId}/uploadImage`

This endpoint accepts multipart form data with an image file and optional metadata. The `petId` must correspond to an existing pet in your store. Supported image formats include JPEG, PNG, and GIF with a maximum file size of 10MB.
